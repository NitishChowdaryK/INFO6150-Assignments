## Description
A single-page stopwatch that times events, associates them with a date, and saves session history to localStorage.

## Features
- Timer display in HH:MM:SS (updates every second)
- Date picker defaults to today
- Event name validation with specific error messages
- Start / Pause-Resume / Stop & Save / Reset controls
- Saves sessions in localStorage and shows most recent first
- Filter session history by date
- Statistics: total sessions count and total time
- Toast notifications + modal dialog (professional UI)

## Technologies Used
- HTML5
- CSS3 (responsive layout)
- JavaScript ES6+
- jQuery
- Async/Await, Promises
- setInterval / clearInterval
- localStorage

## How to Run
Open `stopwatch.html` in a browser and use the controls to time and save sessions.
